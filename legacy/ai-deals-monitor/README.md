# Legacy AI Deals Monitor

This directory contains the archived `AI Deals Monitor` project that previously shared the repository with Health Agent.

Contents include:

- `docker-compose.yml`
- deploy scripts under `deploy/`
- setup and workflow docs under `docs/`
- SQL files under `sql/`
- n8n workflows under `workflows/`
- prompt and service seed files used by that stack

This archive is kept for reference and migration purposes. It is not the active deployment target for the current Health Agent iPhone app.
