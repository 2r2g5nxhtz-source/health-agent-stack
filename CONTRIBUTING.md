# Contributing

Thanks for contributing to Health Agent.

## Ways to Help

- improve the iPhone app
- improve `n8n` workflows
- improve setup docs
- report bugs with reproduction steps
- suggest safer health-data handling patterns

## Development Guidelines

1. Keep changes focused.
2. Prefer local-first and privacy-preserving defaults.
3. Avoid introducing cloud dependencies unless they are optional.
4. Document user-facing behavior changes in `README.md` or `docs/`.

## Pull Requests

Please include:

- what changed
- why it changed
- how it was tested
- any screenshots if the UI changed

## Issues

When opening an issue, try to include:

- device and OS version
- whether the problem is in iPhone app, `n8n`, or deployment
- relevant logs or screenshots
- expected behavior vs actual behavior

## Security

Do not post tokens, webhook secrets, Apple credentials, or private health data in issues or pull requests.
