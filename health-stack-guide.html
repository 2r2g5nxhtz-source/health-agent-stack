Health Stack - установка на MacBook M3 Max

n8n · Ollama · Open WebUI - локально, без подписок

~30 минут $0 навсегда macOS · Apple Silicon базовый терминал
Важно про n8n: твой текущий триал - это облачная платная версия (€24/мес после 14 дней). Self-hosted Community Edition - бесплатна навсегда. Все воркфлоу можно экспортировать из облака и импортировать локально.
1
Homebrew - менеджер пакетов ~2 мин
Шаг 1 - открой Terminal
Cmd + Space → напиши «Terminal» → Enter
Шаг 2 - установи Homebrew
Вставь команду и нажми Enter. Попросит пароль - введи, это нормально.
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
Ждёт ~2 минуты. В конце напишет "Installation successful!"
2
Docker Desktop ~5 мин
Шаг 3 - установи Docker
brew install --cask docker
Шаг 4 - запусти Docker Desktop
Открой через Finder → Applications → Docker. Дождись иконки кита в меню-баре сверху. На все запросы разрешений - Allow.
Шаг 5 - проверь
docker --version
Должно написать: Docker version 27.x.x или выше
3
n8n self-hosted бесплатно навсегда ~3 мин
Шаг 6 - создай рабочую папку
mkdir -p ~/selfhosted/n8n && cd ~/selfhosted/n8n
Шаг 7 - запусти n8n
Скачает образ (~300 MB, один раз) и запустит в фоне.
docker run -d --name n8n -p 5678:5678 --restart unless-stopped -v n8n_data:/home/node/.n8n docker.n8n.io/n8nio/n8n
Флаг --restart unless-stopped означает: n8n запустится автоматически после перезагрузки Mac (если Docker Desktop тоже настроен на автозапуск).
n8n открывается по адресу localhost:5678
4
Ollama - локальный AI ~5 мин
Шаг 8 - установи Ollama
brew install ollama
Шаг 9 - запусти и скачай модель
llama3.2 - лёгкая (~2 GB), быстрая на M3, хорошо понимает русский. Медданные не уходят в облако.
ollama serve & sleep 3 && ollama pull llama3.2
5
Open WebUI - интерфейс к Ollama ~2 мин
Шаг 10 - запусти Open WebUI
docker run -d -p 3000:8080 --add-host=host.docker.internal:host-gateway --restart unless-stopped -v open-webui:/app/backend/data --name open-webui ghcr.io/open-webui/open-webui:main
Open WebUI localhost:3000
6
Управление стеком
Запустить всё (после перезагрузки Mac)
docker start n8n open-webui && ollama serve &
Остановить всё
docker stop n8n open-webui
Посмотреть статус
docker ps
Должны быть видны n8n и open-webui со статусом "Up"
После установки у тебя будет
n8n - автоматизация, Apple Health webhook localhost:5678
Ollama - локальный AI (llama3.2) localhost:11434
Open WebUI - чат с локальным AI localhost:3000
