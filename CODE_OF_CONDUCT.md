# Code of Conduct

## Our Standard

Please be respectful, constructive, and kind.

Examples of positive behavior:

- giving helpful feedback
- assuming good intent
- being patient with beginners
- focusing on the work, not the person

Examples of unacceptable behavior:

- harassment or insults
- discrimination
- bad-faith arguments
- publishing someone else's private information

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and pull requests that violate this code of conduct.

## Scope

This applies in project spaces including issues, pull requests, discussions, and review comments.
